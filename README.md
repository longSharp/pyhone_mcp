# MCP-Zero: 基于Python的MCP协议完整实现

一个完整的Model Context Protocol (MCP)实现，包含服务端、客户端和OpenAI LLM集成，实现了先进的MCP-Zero技术。

## 项目概述

MCP-Zero是一个完整的MCP协议实现，允许你：

- 🚀 构建和运行MCP服务端
- 🔗 创建MCP客户端连接
- 🤖 集成OpenAI语言模型
- ⚡ 实现零样本/少样本的MCP-Zero技术

## 功能特性

### 核心功能
- ✅ 完整的MCP协议实现
- ✅ 服务端工具和资源管理
- ✅ 客户端会话和调用管理
- ✅ OpenAI GPT模型集成
- ✅ MCP-Zero零样本技术
